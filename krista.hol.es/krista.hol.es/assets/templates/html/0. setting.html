Если при работе с WebStorm у вас подвисает интерфейс при том что на Вашем компьютере установленно оперативы больше 2 Гигабайт. Вам следует зайти в папку с установленной программой по умолчанию это путь (C:\Program Files (x86)\JetBrains\WebStorm 143.870.2\bin) найти там файл (WebStorm.exe) открыть его с помощью текстового редактора и в нем исправить строчку

-Xms128m
-Xmx512m
-XX:MaxPermSize=250m

И исправить значение 512 минимум на 1024. После чего тормоза исчезнут.

-Xms128m
-Xmx1024m
-XX:MaxPermSize=250m


Добавляем в файл и сохраняем.

Комментарии

Данный тип комментариев нужно использовать во вне обьектов. В главных функциях.

/**
 * Комментарии вида.
 */

Пример:

public refresh(container: HTMLElement) {
   /** 
	* Первый тип комментариев.
	*/
}

Используются внутри обьектов.

// Комментарии данного вида.

Пример:

public refresh(container: HTMLElement) {
   /** 
	* Используем первый тип комментариев.
	*/
	if (this.value > 1) {
		// Здесь уже используется второй тип комментариев.
	}
}

"layout"

При создании контрола, желательно использовать только высоту контейнера. В этом случае, контрол будет использовать 100% ширины и должен изменяться в зависимости от ширины браузера.

Когда контрол создается он имеет примерно 12 секций в ширину, которые мы можем определять что бы располагать в них те или иные контролы(графики, меню и тд.).

Пример 

"layout": {
	"control": "WebReports.Controls.ReportLayout",
	"rows": [{
		"fluid": "true",
        "cells": [{
			"span": 12,
            "sectionName": "mapCustomControl1"
        }]
    }]
},
